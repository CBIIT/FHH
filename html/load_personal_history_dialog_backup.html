Detailed file opening instructions




To open a file from your computer or portable storage device (for example, a USB flash drive):



    
	
  • If you are opening a file from a portable storage device, make sure the device is properly connected and mounted before starting these instructions
    • 
	
  • 
		In the section above marked 'Open From Computer' - press the Browse... button
	
    • 
	
  • A file selection box will be displayed - locate and select the previously-saved family health history XML file you wish to open, and press OK or Open on the file selection box (depending on the type of computer or web browser you are using)
    • 
	
  • If you are opening a file from a portable storage device, you will need to locate the device in the file selection box to view its contents and then select the file to be opened
    • 
	
  • Press the button marked 'Open From Computer' on this screen
    • 




	Note: Unless it has been renamed on your computer, a saved family health history XML file will usually be FamilyHistory.xml or _FamilyHistory.xml (replace  with the name of the center person, usually you). Note also that only files originally created with My Family Health Portrait are supported for opening.
