Sobre o Meu Retrato de Saúde Familiar

Meu Retrato de Saúde Familiar originalmente foi desenvolvido em colaboração entre o Gabinete da Cirurgião General e o Instituto Nacional de Investigação do Genoma Humano, uma parte dos Institutos Nacionais de Saúde, Departamento de Saúde e Serviços Humanos. A primeira versão do Meu Retrato de Saúde Familiar foi lançada em novembro de 2004 como download de software para o computador. No entanto, funcionava apenas em computadores usando o sistema operacional do Windows, que era uma desvantagem significativa para os usuários do sistema operacional do Macintosh (Mac).

Em novembro de 2005, a ferramenta foi lançada numa versão disponível na internet que usava o navegador de internet no computador pessoal do usuário. Não há um download de software para o computador pessoal do usuário, então a nova ferramenta funciona num computador usando o sistema operacional do Macintosh bem como num computador usando o sistema operacional do Windows. A única exigência é que o usuário tenha uma versão atualizada do software do seu navegador, que é amplamente disponível a partir de várias fontes. O software para o Meu Retrato de Saúde Familiar foi testado em versões atuais do Internet Explorer para Windows, Firefox para Windows e Mac, Netscape para Windows e Mac e Safari para Mac.

Em janeiro de 2009, a ferramenta foi reprojetada e modificada nos seguintes aspectos:

  1. Baseada em Padrões: Aproveitando o trabalho feito pela Comunidade de Informações da Saúde Americana (AHIC), as normas técnicas e de dados centrais foram construídas no desenho de FHH 2.0. Isso significa um aumento de interoperabilidade e uma curva de aprendizado reduzida para os consumidores, profissionais, e investigadores.
  2. Compartilhável: Informações podem ser compartilhadas eletronicamente pelo consumidor com outros membros da família e fornecedores de serviços de saúde. Essa nova funcionalidade permite que os consumidores possam colaborar com familiares eletronicamente para desenvolver um histórico de saúde familiar mais completo.
  3. EHR-Pronto: A versão 2.0 produz arquivos de xml que podem ser integrados facilmente em EHRs existentes com pouca customização ou apoio de TI. É tão simples quanto copiar um arquivo eletrônico de um local para um novo sistema.
  4. Customizável: A ferramenta pode ser baixada e adotada sob a própria marca da organização. Destina-se a ser facilmente adaptável a portais, assim permitindo que as organizações de saúde possam promover a história de saúde familiar sob sua própria marca, usando-a em seu próprio ambiente seguro e contribuir para criar um portal centralizado para obter um número grande serviços para o paciente.
